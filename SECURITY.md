# Security Policy

This repository powers a personal portfolio website. If you find a security
issue in the code here, I'd appreciate a responsible heads-up.

## Reporting a vulnerability

Please report vulnerabilities privately through GitHub:

[**Report a vulnerability**](https://github.com/fortunato/fortunatogeelhoed.com/security/advisories/new)

(Repository → *Security* → *Advisories* → *Report a vulnerability*.)

Private reporting keeps the details out of public issues until a fix is
available. I aim to acknowledge a report within a few days.

Please don't open a public issue for a security problem.

## Scope

In scope: the source in this repository and the build it produces. Out of
scope: third-party services the site integrates with, and findings that
require physical or privileged access to infrastructure I operate.
